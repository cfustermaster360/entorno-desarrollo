Curso de desarrollo web

El desarrollo web es una disciplina que combina diseño, programación y estrategia. Existen diferentes niveles de títulos que ayudan a estructurar la información y facilitan la lectura.

Según Tim Berners-Lee, creador de la web, el HTML fue concebido para dar significado al contenido, no solo para mostrarlo bonito.

En este curso aprenderás a escribir código limpio y semántico. La palabra responsive proviene del inglés y significa adaptable.

El término HTML significa HyperText Markup Language y es la base de todas las páginas web.

Recuerda: escribir buen HTML no es solo una cuestión técnica, sino también de respeto por los usuarios.
