Diario de aprendizaje

24 de enero del 2025

El mes pasado empezamos con HTML creando nuestra primera página web, el nombre del documento fue Hola Mundo.

Aprendimos a crear un documento de HTML5 revisando su head para que tuviera todas las etiquetas correctas, es decir que el lenguaje fuera en español: html lang="es", que tuviera un título: Hola Mundo.

A continuación pasamos a la parte del body y creamos un <h1> para tener un encabezado, un <p> que sería un párrafo y por último un botón.

Lo que más me costó fue la parte del botón por que tenía JS y no me funcionaba

Esto sería el código que usé en la prática: 

<!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Hola Mundo</title>
        <link rel="stylesheet" href="style.css">
        <script src="script.js" defer></script>
    </head>
    <body>
        <h1>Mi Primera Página en HTML</h1>
        <p>Hola <strong>Mundo</strong></p>
        <p><button>Jugador 1</button></p>
    </body>
    </html>

Algo que he aprendido en esta práctica, es un atajo para poder visualizar todo el texto en Visual Studio Code, es decir, que el texto no sea una sóla línea, que lo pueda ver en pantalla todo el texto. El atajo sería el siguiente: Alt + Z
